##########################
欢迎使用 Betalens 量化框架
##########################

**Betalens** 是一个为个人和小型团队设计的量化研究框架，旨在提供一套从数据管理到策略回测和分析的完整、易用的工具链。

无论您是想验证一个简单的因子思想，还是构建复杂的多因子模型，Betalens 都能帮助您快速实现、迭代和评估您的策略。

.. note::

   本文档正在持续更新中。如果您发现任何问题或有改进建议，请随时提出。
